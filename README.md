# 1C Telegram Integration Framework

Репозиторий содержит исходный код программной реализации, подготовленный в качестве приложения к выпускной квалификационной работе.

Тема ВКР: «Разработка фреймворка интеграции платформы „1С:Предприятие“ с мессенджером Telegram для автоматизации процессов учета и отчетности по ресурсам».

## Состав репозитория

- TelegramFramework.bsl — общий модуль Telegram, содержащий основную реализацию фреймворка интеграции: отправку сообщений, обработку входящих команд, маршрутизацию, управление состояниями диалога, работу с текущими и отложенными запросами, inline-клавиатурами, файлами и прикладными сценариями.
- WebhookHandler.bsl — обработчик HTTP-сервиса, принимающий webhook-запросы от Telegram Bot API и передающий их в общий модуль фреймворка.

## Примечание

Из опубликованного кода удалены или заменены конфиденциальные данные: токены Telegram-ботов, внутренние адреса, телефонные номера, идентификаторы и иные сведения, относящиеся к рабочей информационной базе.

Код опубликован для демонстрации архитектурных и программных решений, рассмотренных в выпускной квалификационной работе.
